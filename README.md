## Prerequisites

* Ruby 2.7.1

* PostgreSQL 13.0

* Node 12.18.3

* Yarn 1.22.10


## Installation

```
git clone https://github.com/evgenyneu/computer_code.git
cd computer_code
bundle
yarn install
rails db:create db:migrate
```

## Run local web server

```
rails server
```

Open web site [http://127.0.0.1:3000](http://127.0.0.1:3000).
